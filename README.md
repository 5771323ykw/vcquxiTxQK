# 人事管理系统 java133

## 项目概述

本项目是一个基于Springboot和Vue的人事管理系统，采用Javaee技术栈开发，实现了前后端分离的设计模式。系统采用MVC三层架构，通过Spring、SpringBoot、Mybatis和Vue技术的结合，以及使用Maven作为项目管理工具，打造了一套高效稳定的人事管理系统。

## 技术栈

- **后端**: Springboot、Javaee、Mybatis
- **前端**: Vue
- **数据库**: MySQL
- **项目管理**: Maven

## 功能模块

- 用户管理
- 员工信息管理
- 考勤管理
- 薪资管理
- 权限控制

## 系统角色

- 系统管理员
- 人事专员
- 员工

## 运行环境

- **操作系统**: Windows/Linux/MacOS
- **数据库**: MySQL 5.7+
- **JDK版本**: JDK 1.8+
- **Node版本**: Node.js 12+

## 部署步骤

1. 搭建开发环境，包括Java开发环境和Node.js环境
2. 克隆或下载项目代码到本地
3. 导入MySQL数据库脚本
4. 使用Maven编译后端项目
5. 前端项目使用npm或yarn安装依赖，然后编译打包
6. 部署到Web服务器

## 目录结构

```

    |   |-- src

    |   |-- src

        |-- schema.sql
        |-- data.sql
```

## 核心亮点

- **高效稳定**: 基于成熟的技术栈开发，确保系统运行高效稳定。
- **灵活扩展**: 采用模块化设计，方便系统功能的扩展和维护。
